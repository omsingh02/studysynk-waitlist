## What this is ?

A clean, focused early-access waitlist page for a student project coordination tool. Problem statement, features, audience targeting, trust section, and email capture CTA.

## Sections

1. Hero with early-access badge and CTA
2. The problem — relatable student pain points
3. Features — what the tool does
4. Who it's for — audience cards
5. Trust / early access positioning
6. Waitlist CTA with email form
7. Footer

## Tech stack

- HTML
- CSS (custom properties, mobile-first, violet accent theme)
- No frameworks, no build tools

## Run locally

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```
